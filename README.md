# プログラミング応用
名古屋工業大学情報工学科3年 プログラミング応用講義資料 テンプレートマッチング課題用プログラム
最終テスト用データセット

中の画像はpng圧縮されており，ファイルをデータセットごとにtar.gzで圧縮しています．
ディレクトリ構成は下記になってます．（templateA~Dの名前は可変）
適切にテストプログラムに配置してください．

データセットの10と19はサイズ制限のため前半後半（10_1と10_2など）に分かれています．
```
+ image
+ level1
+ level2
+ level3
+ level4
+ level5
+ level6
+ level7
+ templateA.png
+ templateB.png
+ templateC.png
+ templateD.png
```
